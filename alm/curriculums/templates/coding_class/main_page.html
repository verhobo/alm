CURRENT HOMEWORK:

We've looked at the PARROT PICTURES project on jsfiddle.net. Your goal is to:

  • Notice the layout, and realize that there are three languages that make up the webpage: HTML, CSS, and JavaScript
    • The languages speak to the web application, communication an algorithm: in this case, the problem is that we want to display several large pictures. We want the user to be able to click on buttons to see the pictures, without changing the whole webpage.
    • We use HTML to lay out the basic elements of the page. Example: putting a button on a webpage, and putting a picture on a webpage.
    • We use CSS to define the style of HTML elements. Example: making all buttons green, and all pictures 500 pixels wide.
    • We use Javascript to add events to the webpage. Example: changing the picture when someone clicks the button.
  • Decide upon what you want your picture gallery to be about! Your chickadee sticker business? Cute kittens? Sloths? Minecraft characters?! YOU DECIDE!
