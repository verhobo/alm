Project #1: IMAGE GALLERY

CURRENT HOMEWORK:

We've looked at the PARROT PICTURES project on jsfiddle.net. Your goal is to:

  • Notice the layout, and realize that there are three languages that make up the webpage: HTML, CSS, and JavaScript.
  • The languages speak to the web application, communicating an algorithm: in this case, the issue is that we want to display several large pictures. We want the user to be able to click on buttons to see the pictures, without changing the whole webpage.
  • Our algorithm has these steps: we lay out the page, with a title, buttons to click to change the pictures, and the pictures. Every time the user clicks a button, the Javascript event knows which button was clicked. We hide all the pictures, and then only reveal the one tied to our clicked button.
  • We use HTML to lay out the basic elements of the page. Example: putting a button on a webpage, and putting a picture on a webpage.
  • We use CSS to define the style of HTML elements. Example: making all buttons green, and all pictures 500 pixels wide.
  • We use Javascript to add events to the webpage. Example: changing the picture when someone clicks the button.
  • You've decided already what your gallery is going to be about! Sloths, mushrooms, swords, jellyfish, wolves... you're all doing a great job showcasing your interests, mashaAllah.
